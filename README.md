# ytdlp-scripts
- This is a workflow for tiktok archiving 
- designed on wsl/linux

---

## Method

### The Start
- Create a text file of exact usernames line by line
- use bulk add to create directories and download
- each directory contains username and user id within brackets

### Normal Use
- Run the batch update (cron job or manual, whatever)
- it reads user id and not username, thus is resistant to anything but the below
- Check error log made
  - Account Privated
  - Account banned / limited
  - Your cookies need to be updated
